# Panchayat App

A React Native / Expo application for local Panchayat community use.

## Project setup

This project uses Expo and React Navigation.

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm start
```

Then use the Expo app on a device or an emulator to open the project.

### Helpful scripts

- `npm start` - start Expo development server
- `npm run android` - start Expo and open on Android device/emulator
- `npm run ios` - start Expo and open on iOS simulator
- `npm run web` - run the app in a browser

## Notes

- `node_modules/` is excluded from GitHub because it is generated by `npm install`
- `.expo/` is excluded because it contains local Expo metadata
- Add files to the `assets/` folder if you want media or images tracked by git

## Dependencies

- Expo
- React
- React Native
- React Navigation
- Expo Speech
- Expo AV
- Async Storage
